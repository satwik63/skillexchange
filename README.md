# SkillExchange

A React + Vite skill exchange platform where users can trade skills instead of money. It uses Firebase for authentication, database, storage, and real-time features.

## Features

- Firebase email/password login and signup
- Profile completion prompts and completion percentage
- Editable profile with Storage-backed avatar upload
- Skill-based search and match ranking
- Real-time Firebase chat with timestamps
- Reviews, requests, notifications, and favorites
- Responsive design for mobile and desktop

## Tech Stack

- React + Vite
- Firebase (Auth, Firestore, Storage)
- JavaScript, HTML, CSS
